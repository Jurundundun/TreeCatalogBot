# TreeCatalogBot

TreeCatalogBot - это Telegram бот, предоставляющий пользователю интерфейс для управления каталогом элементов в виде дерева.

## Описание

Этот бот позволяет пользователям создавать, просматривать и удалять элементы в виде древовидной структуры. Он предоставляет удобный интерфейс для управления элементами в древовидном формате через команды в Telegram.

## Функциональность

- Добавление элемента каталога (как корневого, так и дочернего элемента).
- Просмотр всего дерева категорий в структурированном виде.
- Удаление элемента и всех его дочерних элементов из каталога.

## Использование

1. Установите Telegram и найдите бота по имени `@TreeCatalogBot`.
2. Взаимодействуйте с ботом, используя следующие команды:
    - `/addElement элемент` - добавляет новый корневой элемент.
    - `/addElement родитель дочерний` - добавляет дочерний элемент к указанному родителю.
    - `/viewTree` - просмотр всего дерева категорий в структурированном виде.
    - `/removeElement элемент` - удаляет элемент и все его дочерние элементы.

## Запуск

1. Убедитесь, что у вас установлен Java Runtime Environment (JRE).
2. Настройте параметры в `application.properties`.
3. Запустите приложение.

## Технологии

- Java
- Spring Boot
- Hibernate
- TelegramBots API
